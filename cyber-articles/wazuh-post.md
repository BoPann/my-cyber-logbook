# Wazuh

A few weeks ago, I learned about a tool called **Wazuh**—an open-source SIEM and XDR solution. It’s used by large enterprises like **NASA, Salesforce, and Walgreens**.

Naturally, I thought to myself: _Wait… it’s free, and I get to use the same tool as NASA?_ **NASA?!**

So today, I decided to install it and play around with it. And wow—there’s a _lot_ you can do with Wazuh. This write-up focuses on **File Integrity Management (FIM)**, which basically answers one very important question:

**“Whooooo touched my files?”**
