# DNS Record

## Some Common Query Type
- A record - domain:IP4 mapping. I.e, `bopan.me: 192.168.0.1`
- AAAA record - domain:IP6 mapping
- CNAME - domain:domain mapping (subdomain) I.e, `bopann.github.io: bopan.me`

> There are more, but those are I have experienced with.
