# Local and Domain Account

## 1. Overview
In the school or enterprise environment (especially in Microsoft Windows Environment), there are 2 common accounts. They are manged differently. 

## 2. Description
- local account - data are stored locally on the machine. User's data can only be accessed on this host machine only 
- domain account - data are usually managed by the **Domain Controller** which utilized **Active Directory** (serves like a database) 

## 3. Why it Matters?
This is the most common environment that help desk will be in.
